# Personal Curriculum Vitae

This is my CV written in Latex.

## How to compile

Use [Makefile](./Makefile) to compile the CV.

### Useful commands

- Compile the CV (default target)

```bash
make
```

- Compile the CV explicitly

```bash
make all
```

- Clean the output files

```bash
make clean
```
